Community

There are a couple of ways to interact with the K-9 Mail community. Below you'll find the mailing lists:

Support Forum

If you have any questions related to K-9 Mail this forum is the right place to ask.

Dev Talk

For development talk, the developer mailing list is a low-traffic list covering broad development issues.

Apart from the mailing lists, you can find us on IRC and GitHub:

IRC

Many of the developers use IRC to chat in real-time. You can find them on the Freenode network, channel #k-9

GitHub Bug Tracker

Use the issue tracker to report bugs or request new features.
